# Flujo de trabajo del análisis metilómico

## 1. Obtención de datos
Se utilizan datos de microarreglos Illumina correspondientes a muestras de páncreas humano.

## 2. Preprocesamiento
Se realiza limpieza y normalización de los datos para eliminar ruido técnico.

## 3. Análisis diferencial
Se aplican métodos estadísticos para identificar genes o regiones diferencialmente metiladas.

## 4. Interpretación biológica
Se analizan los resultados en el contexto de la diabetes tipo 2.

## 5. Generación de resultados
Se producen tablas y figuras que resumen los hallazgos.
